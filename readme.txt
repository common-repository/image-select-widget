== Description ==

Image Select Widget is a Widget where you can save images in a list and category. With a shortcode you can add a theme to a post or a page. Furthermore you have the possibility to change a lot of settings.

For maximum optic install <a href="http://wordpress.org/plugins/wp-jquery-lightbox/" target="_blank">jQuery Lightbox for Wordpress</a>.

= Translators =

* German (de_DE) - [Marcel Birkholz](http://marcel-online.magix.net/)

== Installation ==

1. Upload `image-select.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[image_select id="1"]` in your page or post

You will find Image Select Widget on the Dashboard in your Wordpress admin panel
